# Student Performance Analysis using Python & SQL

## Project Overview
This project analyzes student academic performance using Python, Pandas, SQLite, and SQL queries. 
Data is processed, analyzed, and visualized to generate meaningful insights.

## Features
- Data creation using Pandas
- SQL JOIN operations
- Performance analysis
- Subject-wise statistics
- Data visualization using Matplotlib

## Technologies Used
Python | Pandas | SQL | SQLite | Matplotlib | Data Analysis

## Key Insights
- Identified top-performing students
- Compared subject performance averages
- Analyzed gender-wise performance trends
